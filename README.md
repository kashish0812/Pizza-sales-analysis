# 🍕 Pizza Sales Analysis Dashboard

## 📌 Project Overview
The Pizza Sales Analysis Dashboard is an Excel-based data analysis project that focuses on understanding pizza sales performance using a structured dataset. The project converts raw sales data into meaningful insights using data cleaning, pivot tables, and dashboard visualization.

The goal of this project is to analyze sales patterns, identify top-performing products, and provide business insights that could help improve decision-making.

---

## 🎯 Project Objectives
- Analyze overall pizza sales performance
- Identify the best-selling and worst-selling pizzas
- Track sales trends over time
- Calculate important business KPIs
- Build a visual dashboard for better understanding of the data

---

## 📂 Dataset Information
The dataset contains detailed information about pizza orders such as:

- Order ID  
- Pizza ID  
- Pizza Name  
- Pizza Category  
- Pizza Ingredients  
- Order Date  
- Order Time  
- Quantity Ordered  
- Unit Price  
- Total Price  

This data helps in analyzing customer ordering patterns and sales performance.

---

## 📊 Key Performance Indicators (KPIs)

The dashboard calculates the following important metrics:

- **Total Revenue** – Total income generated from pizza sales  
- **Total Orders** – Total number of orders placed  
- **Total Pizzas Sold** – Total pizzas purchased by customers  
- **Average Order Value** – Average revenue per order  
- **Average Pizzas per Order** – Average quantity of pizzas in each order  

---

## 📈 Dashboard Insights

### Sales Trends
- Daily and monthly order trends
- Peak order periods

### Category Analysis
- Sales distribution by pizza category
- Popular pizza types among customers

### Best and Worst Sellers
- Top-selling pizzas based on revenue and quantity
- Least popular pizzas

### Revenue Contribution
- Percentage contribution of each pizza category to total revenue
